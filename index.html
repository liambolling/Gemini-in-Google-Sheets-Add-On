How does this work?

Make sure to select the rows in your sheet that you want to send to Gemini. The present values will be sent as prompts, and Gemini will return a value for each. Rows ready for processing will be highlighted in yellow. The batch processing approach helps avoid exceeding the API's rate limit.

Why is this useful?

This feature can be employed for a variety of data processing tasks, such as sentiment analysis or categorization of text. Essentially, it replicates the capabilities of an entry-level analyst at your organization. To understand more about the potential uses of this feature, see our examples.
